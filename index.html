RedSync

Synchronize player data between multiple Spigot/Bukkit servers using Redis.

What is RedSync.

RedSync is a Spigot plugin, that may be interesting for (BungeeCord) networks. RedSync is managing the synchronization of player data across servers using Redis.

What is Redis?

Redis is an in-memory key-value store. It allows developers to easily save serialized objects. If you want to learn more about Redis, start reading here: http://redis.io/topics/faq
